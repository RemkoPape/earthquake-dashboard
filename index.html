How this site is organised

One repository, multiple independent dashboard pages.

Repository structure

Each folder contains one dashboard. This keeps the site clean and makes it easy to embed separate tools in Obsidian.

  • /earthquakes/ for the live earthquake map
  • /volcanoes/ for a future volcano dashboard
  • /weather/ for a future weather or climate dashboard
  • /assets/ for shared images, CSS, or scripts later

Embed example

In Obsidian, embed a specific dashboard with:

<iframe src="https://remkopape.github.io/geo-dashboards/earthquakes/" width="100%" height="700"></iframe>

Built with GitHub Pages. Data sources are loaded live inside each dashboard.
